# US Stock Scanner Pro v1.0

ماسح أولي للأسهم الأمريكية تحت 10 دولارات، يعمل عبر Finnhub ويرسل التنبيهات إلى Telegram.

## ما الذي يفعله؟

- يجلب قائمة الأسهم الأمريكية من Finnhub.
- يفحص دفعة مختلفة في كل تشغيل لتقليل استهلاك API.
- يفلتر السعر بين 0.50 و10 دولارات.
- يقيس الحجم النسبي والزخم المبكر والقرب من أعلى الجلسة.
- يبحث عن محفز خبري إيجابي ويتجنب كلمات التخفيف والعروض.
- يمنع تكرار التنبيه لنفس السهم لمدة 12 ساعة.

## GitHub Secrets المطلوبة

- `FINNHUB_API_KEY`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`

## التشغيل اليدوي

```bash
pip install -r requirements.txt
python -m app.main
```

## تنبيه مهم

هذه أداة رصد وليست توصية مالية. Finnhub المجاني وGitHub Actions لهما حدود سرعة وعدد طلبات؛ لذلك هذه النسخة تفحص السوق على دفعات وليست ماسحًا لحظيًا لكل الأسهم في الوقت نفسه.
